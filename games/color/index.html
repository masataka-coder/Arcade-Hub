遊び方・クリア条件

  • 画面中央のボタンを長押しして流体を落とします。
  • 流体を一番下の「受け皿」に入れ、Target（ターゲット）と同じ色を作りましょう。
  • 光の混色ルール：
    赤 + 緑 = 黄色
    緑 + 青 = シアン（水色）
    赤 + 青 = マゼンタ（紫）
    赤 + 緑 + 青 = 白色
  • ステージ後半では、Cyan(水色)・Mag(紫)・Yel(黄色)の流体も追加されます。組み合わせを考えてみてください！
  • 正しい割合で色が混ざると、画面下のゲージ（Mixing Match Rate）が徐々に上昇します。
  • ヒント：:流体が少ないとうまく混ざりません。思い切って流体を使いましょう！
